# vue-music-player

基于vue和[vue-aplayer](https://github.com/MoePlayer/vue-aplayer)的简单音乐播放器

使用[vant](https://github.com/youzan/vant)作为样式框架

无后端服务器，基于nginx暴露出静态json文件作为数据来源。

献给我们曾经度过的青春。
